# Резюме

С помощью **HTML**, **CSS**, **Bootstrap** вам необходимо создать свое резюме. 
Резюме должно состоять из следующих пунктов:
* ФИО, 
* возраст, 
* опыт работы,
* образование,
* навыки,
* достижения,
* интересы.
